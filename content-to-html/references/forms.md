# 形制库

不是菜单，是"提醒你可以多样"的开口。每选一种就钻进它的内在一致性。

## 选择原则

- 让内容选形制，不要让形制找内容
- 说"现代极简"不算选择；说"瑞士平面设计的杂志内页"才算
- 具体名字让你被风格的内在一致性拽住

## 编辑/出版物类

| 形制 | 视觉特征 | 适合内容 |
|---|---|---|
| 文学杂志（The Paris Review / n+1） | 衬线为主，大字号 italic，§ 节标记，边距数字 | 深度访谈、文化评论 |
| 学术期刊 | 高密度衬线，脚注感，行号，ABSTRACT/KEYWORDS 框 | 研究方法论、技术深潜 |
| 70 年代出版物 | OKLCH 暖纸 + 朱砂红，印刷雕版感 | 商业洞察、行业观察 |
| 中文文人出版物 | 宋体 + 竖排标题或 spine，§ 节标记 | 人文思辨、哲学 |
| 独立杂志 Zine | 手写体 + 拼贴 + 不规则网格 | 创意分享、个人故事 |

## TOC 形态与形制的对应

| 形制方向 | TOC 推荐形态 | 展开策略 |
|---|---|---|
| 文学出版物 | 右侧 spine 式数字标记 | 始终展开（spine 是视觉核心） |
| 工程蓝图 / Mission Control | 左侧固定窄栏 | 默认收起章节号，hover 展开 |
| 学术教材 | 顶部吸附 tab bar | 始终展开 |
| 美术馆 catalog | 圆形 dock / 角落悬浮章节号 | 滚动时浮现，空闲时淡出 |
| Brutalist | 边栏可折叠抽屉 | 只显示进度圆点，点击弹出 |
| 暗金沉浸 | 底部浮动胶囊 | 滚动时浮现 |

**移动端一律折叠**：< 1120px 改为顶部按钮 / 抽屉触发。
