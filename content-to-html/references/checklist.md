# 质量检查清单

P0 未过不交付。P1 应当修复。P2/P3 建议改进。

---

## P0 · 必杀（命中任何一条 = 改稿）

### 基础设施缺陷
- [ ] 无常驻 TOC 导航
- [ ] TOC 无 IntersectionObserver 实时高亮
- [ ] 无响应式断点（至少 `@media (max-width: 880px)` 一档）
- [ ] 小屏 TOC 强占主内容视野（未折叠）
- [ ] CSS 未使用变量（全是 hardcode 颜色值）
- [ ] section 无 id（TOC 无法锚定）

### AI Slop 视觉
- [ ] 紫色渐变在白底上
- [ ] 使用 Inter / Roboto / Arial 字体
- [ ] Hero + 三栏 features + CTA + footer 的落地页结构
- [ ] 无任何视觉签名（干净到没特征）
- [ ] 配色全用 60% 灰度 + 一个蓝色 accent
- [ ] Material Design / shadcn 默认圆角卡片 + 阴影

### 内容缺陷
- [ ] 口语直搬上去（违反反向工程原则）
- [ ] 无交互组件（或交互只是 hover 变色）
- [ ] 章节标题写成 "01. The First Chapter" 式 AI 味命名

---

## P1 · 应修复

### 结构
- [ ] 章节数不在 5-8 范围
- [ ] 无 Self-check / Takeaways 模块
- [ ] 无 Coda（结尾金句）
- [ ] 无 Colophon（设计说明）
- [ ] 文字密度均匀分布，无大字号视觉锚点

### 设计一致性
- [ ] TOC 样式与全篇形制脱节（如出版物形制用了通用灰色列表）
- [ ] 视觉签名与形制矛盾
- [ ] 字体选择与形制不匹配（如工程蓝图用了衬线体）
- [ ] Cover 缺少元数据（来源/时长/章节数/日期）

### 交互
- [ ] 交互无实际信息密度（不改变信息的展示）
- [ ] 交互在移动端不可用

### 动效节奏
- [ ] 无 scroll reveal（内容一次性全部可见，无层次感）
- [ ] 无入场编排（Cover 元素同时出现，无仪式感）
- [ ] 无 hover 微交互（卡片/列表项悬停无反馈）
- [ ] 章节之间无视觉转场（直接拼接，无呼吸感）
- [ ] 缺少 `prefers-reduced-motion` 降级

---

## P2 · 建议改进

- [ ] 代码行数 < 800 或 > 1500
- [ ] Google Fonts 加载超过 4 个字体家族（影响性能）
- [ ] 无进度条
- [ ] 无平滑滚动
- [ ] 色板与上一份产物色相区重叠（Hue 差 < 60°）
- [ ] 字体家族与上一份产物重复
- [ ] "Built with ❤️" / "Get Started" / "Learn More" 等 SaaS 套话
- [ ] 无意义 emoji 装饰（✨🚀💡）
